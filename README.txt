
The plastex-docbook project contains templates and classes 
to enable the plasTeX Python package to render documents written in
LaTeX markup language to DocBook 4.5 and DocBook 5 XML.

To use this package you must install the plasTeX renderer available at
http://plastex.sourceforge.net/

To install plastex-docbook:
- download
- upzip
- ``python setup.py install``


Once installed, you can invoke plastex using one of the following renderers:
--renderer=DocBook45 
or 
--renderer=DocBook50
